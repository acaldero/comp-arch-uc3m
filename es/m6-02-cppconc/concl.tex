\section{Conclusión}

\begin{frame}{Resumen}
\begin{itemize}
  \item C++ ofrece un modelo de concurrencia mediante una combinación
        de lenguaje y biblioteca.
  \item La clase \cppid{thread} abstra un hilo de SO.
  \item La sincronización se consigue combinando \cppid{mutex} y \cppid{condition\_variable}.
  \item \cppid{std::async} ofrece un mecanismo de alto nivel para ejecución de tareas.
  \item \cppid{std::future} permite transportar resultados y excepciones entre hilos.
\end{itemize}
\end{frame}
